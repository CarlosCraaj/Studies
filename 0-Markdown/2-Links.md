# Links

> Criando Links para uma seção no mesmo arquivo ou em outro arquivo.

----

Para criar Links, usa-se `[Texto/Botão](Endereço/Link)`

## Exemplo:

[Cabeçalho](#cabeçalho---titulo) <br>
[PHP](../PHP/if.php) <br>
[Google - Obrigado por existir.](https://google.com)
