# Ênfase

Você pode adicionar ênfase como: negrito, itálico e riscado.  

```
Negrito - Markdown:
**Exemplo Negrito**
__Exemplo Negrito²__

----
Itálico - Markdown:
*Exemplo Itálico*
_Exemplo Itálico²_

Negrito e Itálico - Markdown:
***Exemplo 1***  
___Exemplo 2___  
__*Exemplo 3*__  
**_Exemplo 4_**

Texto Riscado - Markdown
~~Exemplo~~
```
