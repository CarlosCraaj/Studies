# Lista Ordenada Simplificada
Para fazer uma lista ordenada simplificada, usamos `00. Exemplo`.  
**Não Funciona com indentação**

```
Markdown:
00. Item 1
00. Item 2
00. Item 3
```

00. Item 1
00. Item 2
00. Item 3

----

Para que o primeiro valor não seja 0, usamos `01.` em vez de `00.`.
```
Markdown:
01. Item 1
00. Item 2
00. Item 3
```

01. Item 1
00. Item 2
00. Item 3
